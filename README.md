## About

Personal blog originally hosted at bouncerblog.com and swantron.com, now converted to a static site using Hugo. Contains 1,039 posts spanning from 2005 to present.

## Tech Stack

- **Hugo** - Static site generator
- **Paper Theme** - Clean, minimal theme
- **GitHub Pages** - Free hosting
- **GitHub Actions** - Automated deployment

## Features

- WordPress-compatible permalinks
- Featured images for posts
- RSS feed
- JSON API output
- Dark mode support

## Local Development

```bash
cd hugo-site
hugo server
```

Visit http://localhost:1313

For detailed documentation, see [`hugo-site/README.md`](hugo-site/README.md).
